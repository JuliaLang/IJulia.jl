Changelog

This documents notable changes in IJulia.jl. The format is based on Keep a Changelog.

Unreleased

Added

  • installkernel() now supports a displayname argument to customize the kernel display name (#1137).

Fixed

  • The internal heartbeat thread will now shut down cleanly (#1135, #1144). This should prevent segfaults upon exit.
  • Various fixes to the messaging code to be compliant with Jupyter (#1138).
